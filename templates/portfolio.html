Freeth - A Social Network for Passionate People

Jan 2016 - December 2016

A social networking site that connects passionate people based on their interests and location.

Full Stack development was done for this Single Page application. The entire front-end was coded using HTML5, PostCSS and BackboneJS. The site is completely responsive and works on both tablet and mobile screens too.

The backend and the REST API was built using the Python Flask framework and the powerful API creation framework called Falcon.

For the database, we used both PostgreSQL and Titan (graph database) to define a well-structured model for the data and their relationships. Celery and RabbitMQ were used for managing tasks like sending emails and building the database index.
